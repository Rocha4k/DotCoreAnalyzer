How is the DotCore Score calculated?

The Overall Score is the arithmetic mean of 5 independent categories, each with a maximum of 100 points.

Performance + Security + SEO + Usability + Quality / 5 = Overall Score

How to compare with other tools?

Each tool uses a different methodology. The DotCore score does not replace specialised tools — it is a holistic, quick view. Here is what each one analyses:

G
Google PageSpeed Insights Focus: Real browser rendering performance (Core Web Vitals: LCP, CLS, INP). Uses real Chrome user data. Does not analyse security or accessibility.
GT
GTmetrix Focus: Load performance (Waterfall, LCP, TBT). Based on Lighthouse + WebPageTest. Does not analyse SEO, security or accessibility.
AH
Ahrefs / SEMrush Focus: Technical and content SEO, backlinks, keyword tracking. Use full crawlers with access to backlink databases. Do not analyse live performance.
Mz
Mozilla Observatory Focus: Security only (HTTP headers, cookies, CSP, HSTS, SRI). Identical to our "Security" category. Does not analyse performance, SEO or accessibility.
W
WAVE / axe Focus: Accessibility (WCAG 2.1). Requires real JS rendering. Analyses colour contrast, keyboard navigation, ARIA semantics. Much deeper than DotCore in this area.
DC
DotCore Analyzer — you are here Focus: 360-degree rapid view. Analyses network performance, HTTP security, technical SEO, base accessibility, DNS forensics and infrastructure — all in one tool, with no account needed across 5 services.

Limitations & Confidence Level

High confidence
  • TTFB measured in real time
  • HTTP Headers (exact, from server)
  • DNS / IP / Geolocation
  • Sensitive file exposure
  • SEO meta tags (title, desc, og:*)
  • Cookies (Secure/HttpOnly)
Medium confidence
  • Tech stack detection (based on HTML/header patterns)
  • Broken links (limited to 20 links)
  • Keyword density (excludes JS-generated content)
  • Alt ratio (static HTML images only)
Not analysed
  • Core Web Vitals (LCP, CLS, INP) — requires real browser
  • Colour contrast (requires rendering)
  • JS-generated content (SPAs)
  • Backlinks and domain authority
  • Image/video load speed
